# 🎉 Eventara - Event Management Website

## 📌 Project Overview

Eventara is a modern and responsive Event Management Website designed to help users discover, explore, and register for events with ease. The platform provides an intuitive interface for browsing upcoming events, registering for participation, and contacting organizers.

This project demonstrates front-end web development skills using HTML, CSS, and JavaScript while focusing on responsive design and user-friendly interaction.

## 🚀 Features

### 🏠 Home Page

* Attractive hero section
* Event statistics dashboard
* About section
* Upcoming events showcase
* Event countdown timer
* User testimonials
* Call-to-action section

### 📅 Events Page

* Upcoming event listings
* Event categories
* Event search functionality
* Responsive event cards
* Quick registration access

### 📝 Registration Page

* Event registration form
* User information collection
* Event selection dropdown
* Form validation
* Registration success notification
* Local Storage support

### 📞 Contact Page

* Contact information section
* Contact form
* FAQ section
* Success message on submission

### 🌙 Additional Features

* Dark Mode Toggle
* Responsive Design
* Local Storage Integration
* Smooth Animations
* Mobile-Friendly Layout

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API
* Font Awesome Icons

## ⚙️ Installation and Setup

### 1. Clone the Repository

bash
git clone https://github.com/your-username/eventara.git

### 2. Open the Project Folder

bash
cd eventara


### 3. Run the Project

Open `index.html` directly in your browser

OR

Use Visual Studio Code with the Live Server extension:

1. Open the project folder in VS Code
2. Install the Live Server extension
3. Right-click on `index.html`
4. Select **Open with Live Server**
